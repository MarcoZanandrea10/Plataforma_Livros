# Página Marcada

Site simples para gerenciamento de livros pessoais, permitindo cadastrar, organizar e acompanhar o progresso das leituras.

## Sobre o projeto

O **Página Marcada** é uma aplicação web feita com HTML, CSS e JavaScript puro.  
O objetivo do sistema é permitir que o usuário cadastre seus livros, acompanhe o andamento da leitura e organize tudo por categorias/status.

O projeto usa `localStorage`, então os dados ficam salvos no próprio navegador.
